# Radio photos — drop images here for the supplies page

Copy your photos into this folder with these names (HTTPS required for eBay):

| Filename | Unit | Serial |
|----------|------|--------|
| `cm300-922tmc5022.jpg` | Motorola Radius CM300 | 922TMC5022 |
| `cm200-922tmu3381.jpg` | Motorola Radius CM200 | 922TMU3381 |
| `cm200-922tne0190.jpg` | Motorola Radius CM200 | 922TNE0190 |
| `cm200d-751tgq0925.jpg` | Motorola CM200d | 751TGQ0925 |

After adding files, set `imageUrl` in admin to:
`https://prince-george-transport.pages.dev/img/radios/<filename>`

Then run `npm run deploy`.
